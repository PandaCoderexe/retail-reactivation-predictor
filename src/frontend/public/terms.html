Terms of Service

Prediction Accuracy

This application provides machine learning predictions for customer reactivation analysis. Predictions can be incorrect and should not be treated as guaranteed outcomes.

Business Decisions

Model output should support business judgment, not replace it. Review the source data, model limitations, and commercial context before acting on any prediction.

Uploaded Data

Only upload JSON or CSV data that you are allowed to process. Files selected in the browser are read locally and sent to the configured backend when you submit a prediction.

Local Development

This tool is configured for a local backend and model API during development. Confirm environment variables and deployment settings before using it outside local testing.
